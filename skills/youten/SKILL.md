# youten: 要点

コピペ不可資料も含めて要点を抽出。ターミナル出力のみ、ファイル保存なし。入力言語に合わせる (en→en, ja→ja)。

## 入力

- パス (.pdf/.png/.jpg/.md/.txt) → Read で読む
- URL → WebFetch
- 文章貼り付け → そのまま
- 引数なし → 「何の要点ですか?」と短く返す

## 形式 (固定3項目)

1. **これは何** — 一行
2. **中身 / 決まったこと** — 1〜3項目、短い引用付き
3. **次の動き / 未決** — 1〜3項目

## ルール

- 専門用語は日常語に翻訳 ("cache" → "ためておく")
- 引用は短く「鉤括弧」/ "short"
- 自信なしは「だいたい」「可能性が高い」と明記
- 15行以内
- 見出し `#`、メタ発言、丸ごとコピー、ファイル保存は禁止
