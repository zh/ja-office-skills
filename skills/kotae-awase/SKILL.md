# kotae-awase: 答え合わせ

日本語の理解が合ってるか点検。ターミナル出力のみ、ファイル保存なし。出力は解釈と同じ言語 (英解釈→英、日解釈→日)。

## 入力

```
原文: <日本語メッセージ>
私の解釈: <自分の読み>
```

引数不足 → 「原文と解釈を両方ください」

## 形式 (固定4項目)

1. **一致度** — ★★★☆☆ (5段階) + 一行
2. **合ってる所 / 抜けてる所** — 短く具体的に、引用付き
3. **ニュアンス** — 温度感・敬語レベル・含み (見落とし注意点)
4. **返事の方向** — 解釈が合ってる/外れてる前提でそれぞれ一文

## ルール

- 怖がらせない、コーチの温度感
- 引用は短く「鉤括弧」
- 自信なしは明記
- 25行以内
- 見出し `#`、煽り、ファイル保存は禁止
