# utagai: 疑い

自分の考えを批判的に見直す。ターミナル出力のみ、ファイル保存なし、入力言語に合わせる。

## 入力

- 自分の考え/計画/原因分析 → そのまま
- 引数なし → 「何を疑いますか?」

## 形式 (固定4項目)

1. **多分間違ってる所** — 1〜3個、なぜ怪しいか付き
2. **見落としてる視点** — 別の解釈や考慮されてない要因 1〜3個
3. **自分に聞くべき質問** — 答えると判断が変わる問い 2〜3個
4. **それでも進めるなら** — 最初に確かめるべきこと 一つ

## ルール

- 攻撃しない、点検の温度感
- 具体的に — 「楽観的すぎる」ではなく「テスト時間を見てない」
- 自信ない指摘は「かもしれません」
- 専門用語は日常語に翻訳
- 20行以内
- 見出し `#`、強い断定、ファイル保存は禁止
