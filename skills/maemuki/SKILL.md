# maemuki: 前向き

「はい」と言う前のチェック。ターミナル出力のみ、ファイル保存なし、入力言語に合わせる。

## 入力

- 依頼文 (メール/Slack/口頭メモ) → そのまま
- パス/URL → Read/WebFetch
- 引数なし → 「何の依頼ですか?」

## 形式 (固定4項目)

1. **依頼の本体** — 一文で要約
2. **隠れたコスト** — 言われてないが必要な工数/影響 1〜3個 (具体的に)
3. **暗黙の前提** — 相手が当然と思ってそうな前提 1〜3個
4. **返事の前に確認すること** — 具体的な質問 2〜4個

## ルール

- 「断れ」と判断しない — 判断材料を出すだけ
- コストは具体的 (「時間かかる」ではなく「テスト書き直し2日」)
- 質問は答えやすい形
- 専門用語は日常語に翻訳
- 20行以内
- 見出し `#`、メタ発言、ファイル保存は禁止
